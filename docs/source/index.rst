Welcome to NL2UniProt's documentation!
===================================

**NL2UniProt** is a Python library that leverages multimodal deep learning to enable queries for protein sequences
in UniProt database with natural language descriptions with a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
